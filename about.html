The purpose of Marshall tools is to facilitate the construction of high-level graphics for users not familiar with programming languages.All tools are and will ALWAYS be free. The applications are currently in the testing phase, if you find any problems during use, please contact us via marcel.ferreira@unesp.br.

How to use

Currently it is possible to check the gene names for humans, mice and rats using the following identifiers: SYMBOL ,ENTREZID and ENSEMBL.

Identifiers must be typed separated by SPACES or ENTERS. If the following error message appears written in red "None of the keys entered are valid keys for 'SYMBOL'. Please use the keys method to see a listing of valid arguments.", it means that the first gene searched for was not found in the database and that it was probably written incorrectly. When the search contains multiple genes, those that are incorrect have blank lines for the remaining columns.

How to format gene symbols

Greek letters are never used, as well as hyphens.Gene symbols should be italicized (When referring to proteins, the same symbols should be used, but not italicized).For human genes the symbols should be written in capital letters, while for mouse and rat only the first letter should be capitalized.

EXAMPLE:

Human: BGLAP

Mouse: Bglap

Rat: Bglap
